# Streamlit Financial Analyzer

## Overview
The Streamlit Financial Analyzer is a web application designed to provide financial analysis based on uploaded financial data. It allows users to upload CSV, XLSX, or XLS files containing financial metrics and generates a comprehensive analysis, including a financial health score, summaries, recommendations, and visualizations.

## Features
- Upload multiple financial documents (up to 5 files) in CSV, XLSX, or XLS formats.
- Analyze financial data to calculate key metrics and generate a financial health score.
- Display financial summaries, strengths, weaknesses, and recommendations.
- Visualize financial metrics using interactive charts with Plotly.
- Support for multiple languages and financial terminology.

## Requirements
To run this application, you need to have the following Python packages installed:
- streamlit
- pandas
- openpyxl
- plotly

You can install the required packages using the following command:
```
pip install -r requirements.txt
```

## Running the Application
To run the Streamlit application, navigate to the project directory and use the following command:
```
streamlit run app.py
```
